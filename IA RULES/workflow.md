# Claude Code Workflow

Claude must follow a strict development workflow.

---

# Step 1 — Request Analysis

Claude analyzes the request and identifies:

- type of website
- business category
- features needed

---

# Step 2 — Engineering Questions

Claude asks technical questions about:

- backend
- database
- hosting
- APIs
- integrations

---

# Step 3 — Architecture Proposal

Claude proposes a system architecture.

Example:

Frontend: Next.js  
Backend: Node.js  
Database: PostgreSQL

---

# Step 4 — Confirmation

Claude must wait for user confirmation before coding.

---

# Step 5 — Implementation

Only after confirmation may Claude begin generating code.

---

# Step 6 — Testing and Review

Claude verifies:

- code clarity
- maintainability
- performance
- simplicity for local business owners
